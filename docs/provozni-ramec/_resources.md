# Master adresář zdrojů — Provozní rámec Ikaros (v1.0)

Ověřené zdroje pro provozní/compliance knihu. Status: `ověřeno` = existence + URL + charakter potvrzeny přímým fetchem; `pravděpodobně` = existuje, detail nepotvrzen; `doplnit` = agent při psaní kapitoly dohledá a ověří primární zdroj. Termíny a čísla předpisů se před tiskem ověřují proti konsolidovanému znění.

Adresář akcí, conů, heren, médií a DDM je v `docs/komunitni-strategie/_resources.md` — sem se nekopíruje, jen odkazuje (Část VI, kapitola o prezentačních kanálech).
