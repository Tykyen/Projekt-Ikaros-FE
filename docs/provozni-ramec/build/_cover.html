Provozní rámec platformy — web a jeho minimum, GDPR · DSA · AI Act, obsah a licence, foto a prezentace v komunitě, etický kodex a podmínky serveru. Vše, co web potřebuje ke spuštění a ukázání komunitě.
Sesterská kniha k Právnímu rámci platformy — zatímco Právní rámec řeší „co vlastníme a co nás chrání", Provozní rámec řeší „jak web provozujeme, jak ho píšeme a jak ho bezpečně ukazujeme".
