# Master adresář zdrojů — Komunitní strategie Ikaros (v1.0)

Vyčerpávající, deduplikovaný adresář ověřených online zdrojů z paralelního web fact-checku (8 okruhů, červenec 2026). Status: `ověřeno` = existence + URL + charakter potvrzeny; `pravděpodobně` = existuje, ale detail (termín/aktuálnost) nepotvrzen přímým fetchem; `neověřeno` = nedohledán oficiální zdroj. Termíny akcí jsou k roku 2026 a před tiskem se ověřují u pořadatele (viz `_ledger.md`, sekce B).
