# Zvuková databáze

**TL;DR:** Databáze YouTube zvuků (ambient/SFX) s metadaty — světové zvuky lze nominovat do globální knihovny, kterou schvaluje Admin.

## Detail

Zvuková databáze (14.4) drží odkazy na zvuky (YouTube) s metadaty pro atmosféru hry:
- **světový zvuk** — privátní zvuk světa, nominovatelný do globálu,
- **globální zvuk** — schválený, importovatelný do všech světů,
- **nominace** — PJ nominuje zvuk do globálu, Admin schválí/zamítne.
