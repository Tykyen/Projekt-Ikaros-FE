# Hexová mřížka

**TL;DR:** Šestiúhelníková mřížka [[takticka-mapa|taktické mapy]] s axiálními souřadnicemi (q, r) pro pozice a výpočty.

## Detail

Mapa stojí na hexové (šestiúhelníkové) mřížce. Pozice [[token|tokenů]] a [[efekt-na-mape|efektů]] se adresují axiálními souřadnicemi `(q, r)`. Hexy řídí dosah pohybu, sousednost a výpočty vzdáleností.
