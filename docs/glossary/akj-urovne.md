# AKJ úrovně

**TL;DR:** Stupně utajení informací na stránkách [[svet|světa]] — [[pan-jeskyne|PJ]] řídí, co hráči vidí, a může postupně odhalovat skryté pasáže.

## Detail

Některé informace ve světě jsou pro hráče **úmyslně skryté** (např. tajné poznámky o NPC, plot twisty, mapa neprozkoumaného území). PJ (nebo [[pomocny-pj]]+) může:

- označit pasáže obsahu AKJ úrovní → ostatní hráči je nevidí,
- později obsah **změnit / doplnit** a **odhalit** ho hráčům.

AKJ je tedy mechanika gradování informací během hry — PJ má kontrolu nad tím, co a kdy se odkryje.

⚠️ **Definice TBD** — konkrétní stupně AKJ úrovní (kolik jich je, jak se značí, jak se odhaluje hromadně, význam zkratky „AKJ") je třeba doplnit z technické dokumentace nebo specs.
