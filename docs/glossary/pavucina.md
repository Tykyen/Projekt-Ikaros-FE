# Pavučina

**TL;DR:** Síť vztahů mezi subjekty světa (NPC, frakce, místa) s příběhovými linkami — vizuální graf + dashboard pro PJ.

## Detail

Pavučina (campaign) je nástroj PJ pro správu příběhových vztahů. Skládá se z:
- **subjektů** — uzly: [[npc|NPC]], frakce, místa,
- **vztahů** — vazby mezi subjekty se statusem (např. krize),
- **storylines** — příběhové linky seskupující subjekty a jejich vývoj.

Krizové vztahy se zvýrazňují na grafu i dashboardu.

⚠️ Pozn.: tahle „pavučina" = campaign graf vztahů. Nezaměňovat s „wood-wide web" (`isWoodWide`) z jiné části systému.
