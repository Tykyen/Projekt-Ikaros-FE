# Typ stránky

**TL;DR:** Druh [[stranka|stránky]] určující její chování a layout — Lokace, Noviny, Seznam, Galerie, Zoom, Rodokmen, Obrazovka, Ostatní, Postava, NPC.

## Detail

Typ stránky volí, jak se stránka renderuje a co umí. Mezi typy patří: Lokace ([[lokace]]), Noviny, Seznam, Galerie, Zoom, Rodokmen, Obrazovka, Ostatní (klasická wiki), Postava ([[postava]]), NPC ([[npc]]).

Některé typy jsou navázané na [[postava|Character]] (`characterRef`) — např. Lokace/Postava/NPC.
