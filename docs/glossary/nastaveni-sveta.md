# Nastavení světa

**TL;DR:** Soubor parametrů [[svet|světa]] (`WorldSettings`) — motiv, skupiny, AKJ úrovně, měny, šablony, emoty, kalendáře, hlavní lišta.

## Detail

Nastavení světa je centrální konfigurace, kterou spravuje [[pan-jeskyne|PJ]]. Zahrnuje mj.:
- [[tema|motiv]] a [[theme-overrides|theme overrides]],
- skupiny (družiny) a jejich znaky (`groupImages`),
- [[akj-urovne|AKJ úrovně]] utajení,
- [[mena|měny]], [[obchod|obchodní]] šablony,
- vlastní [[vlastni-emote|emoty]],
- [[kalendar-sveta|kalendáře]] (vč. samostatného timeline kalendáře),
- [[hlavni-lista|hlavní lištu]] (menu builder).
