# Matrix rain

**TL;DR:** Speciální vizuální efekt skinu „ikaros" — padající znaky (matrixní déšť) jako součást brand identity platformy.

## Detail

Matrix rain je JS animace padajících znaků, charakteristická pro [[skin|skin]] „ikaros" (fialové synthwave). Je to skin effect — dekorativní vrstva nad pozadím, součást značkové identity Ikaros.

- v dokumentaci: [10-nastaveni-hlavni-lista.md](docs/funkce/10-nastaveni-hlavni-lista.md)
- v UI: pozadí skinu ikaros
