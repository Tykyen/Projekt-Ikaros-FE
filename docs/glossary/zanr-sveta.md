# Žánr světa

**TL;DR:** Předvolba žánru [[svet|světa]] (11 možností: fantasy, dark fantasy, sci-fi…) — odvozuje výchozí [[tema|motiv]].

## Detail

Žánr je atmosférická předvolba volená při zakládání světa (~11 možností). Z žánru se odvozuje výchozí [[tema|motiv]] vzhledu. Je to čistě nálada/styl — nepleť s [[herni-system|herním systémem]] (pravidla) ani [[technologicka-uroven|technologickou úrovní]] (vyspělost techniky).
