# Úvodní strana světa

**TL;DR:** Vstupní dashboard [[svet|světa]] (krok 5.2) ve třech sloupcích — akce / [[novinky]] / [[oblibena-stranka|oblíbené stránky]].

## Detail

Úvodní strana světa je rozcestník po vstupu do světa. Layout třech sloupců:
1. **akce** — rychlé akce + nadcházející [[herni-akce|herní akce]],
2. **[[novinky]]** — oznámení vedení,
3. **[[oblibena-stranka|oblíbené stránky]]** — osobní zkratky.
