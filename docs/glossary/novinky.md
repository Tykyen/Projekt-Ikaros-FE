# Novinky

**TL;DR:** Veřejná oznámení [[svet|světa]] s komentáři — zobrazují se na úvodní straně světa.

## Detail

Novinky jsou oznámení vedení světa pro členy (s možností komentářů). Vystupují na úvodní straně (dashboardu) světa jako jeden ze tří sloupců (akce / novinky / oblíbené stránky, krok 5.2).

Vytvoření Ikaros novinky může spustit notifyAll (push) — viz web-push.
