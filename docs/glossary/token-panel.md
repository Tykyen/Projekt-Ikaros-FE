# Token panel

**TL;DR:** Boční panel s detaily [[token|tokenu]] na [[takticka-mapa|mapě]] (staty, HP, deník) — lze ukotvit (dock), volně přesouvat (drag) nebo nechat jako overlay.

## Detail

Token panel zobrazuje informace o vybraném [[token|tokenu]]: [[staty-systemu|staty]], HP, embed [[denik-postavy|deníku]]/poznámek. Režimy zobrazení: ukotvený (dock), tažený (drag), překryvný (overlay).
