# Hero obrázek

**TL;DR:** Hlavní velký obrázek [[svet|světa]] / [[stranka|stránky]] / [[herni-akce|akce]] s nastavitelným výřezem (focal point, zoom, fit).

## Detail

Hero obrázek je dominantní vizuál v hlavičce. Podporuje nastavení výřezu: focal point (kam se má obrázek centrovat), zoom a režim fit. Tím vypadá dobře na různých poměrech stran (mobil i desktop).
