# 14.7 Export světa — shrnutí práce a stavu

Popis toho, co bylo na kroku 14.7 odvedeno, v jakém je to stavu a co zbývá. Detailní spec: [spec-14.7.md](spec-14.7.md). Plány: [plan-14.7a.md](plan-14.7a.md), [plan-14.7b.md](plan-14.7b.md), [plan-14.7c.md](plan-14.7c.md). Chyby cestou: [chybový deník — tisk](../../chybovy-denik/tisk.md).

## Co je 14.7

Export světa = dva pilíře:
- **Pilíř A — Tisk / PDF** jednotlivých entit (FE, `window.print`). Body 2–14 zadání (deník, záložky, kalendář, bestiář, mapy, hvězdná, pavučina, storyboard, obchod, stránka).
- **Pilíř B — Záloha celého světa** do ZIP + import (BE). Bod 1 zadání.

## Stav po částech

### Pilíř A — tisk (FE)
- **14.7a** ✅ framework (`printModeAtom`, `usePrint`, `PrintButton`, `print.css`) + tisk stránky a deníku PC/NPC.
- **14.7b** ✅ rozšíření na záložky, kalendář s rozsahem, bestiář, mapy-atlas, hvězdná (DOM seznam), pavučina, storyboard, obchod.
- ⚠️ **Funkčně rozpracováno — vizuální tisk se ladí.** Při reálném testu na produkci se ukázala série problémů (prázdné stránky, chybějící obrázky, mizející záložky, rozbitý vzhled). Postupné pokusy a jejich selhání jsou zaznamenány v [chybovém deníku](../../chybovy-denik/tisk.md) (CH-001…008 + CH-META).
- **Aktuální mechanismus:** tisk přes samostatné okno (`window.open`) — text se tiskne, ale věrný vzhled + obrázky nejsou spolehlivé. **Rozhodnuto přehodnotit přístup** (serverové PDF / dedikovaná tisková šablona) — viz CH-META.

### Pilíř B — záloha (BE), commitnuto+pushnuto
- **14.7c export ✅** — BE modul `world-export` (`GET /worlds/:worldId/export`), ZIP (`manifest.json` + `data.json`) celého lore stromu světa. Commity `59c552d`, `9cf98be`.
- **Scope:** `pj-full` (PJ / platform Admin čte vše). **Hráčský `viewer-partial` = 403** (rozhodnuto — leak-safe, hráč si tiskne přes pilíř A).
- **Obsah:** svět + nastavení + členové, stránky, postavy + všechny subdocs (deník/finance/výbava/poznámky/kalendář/účty), kalendáře, taktické scény, atlas map + složky, hvězdná mapa, timeline, události, bestiář světa, kampaň (pavučina/storylines/scénáře/obchod), per-PJ poznámky, volitelně chat.
- **Média:** binárky se stahují do `media/` (graceful skip propadlých); URL zůstávají v datech.
- **FE:** tab „Export / Záloha" v nastavení světa (PJ-only) + upozornění, že příprava může trvat několik minut.

## Vědomá rozhodnutí
- **Import (B5/B6) ODLOŽEN** — formát je import-ready (stabilní ID + `version` + `scope`), import se doplní později jako vlastní krok.
- **Hráčský export = 403** (jen PJ/Admin) — leak-safe; reuse filtrů napříč ~15 moduly je samostatný rizikový kus.
- **PDF u tisku = přes prohlížeč** (`window.print` → „Uložit jako PDF"), ne serverový render — toto se ale u věrného vzhledu ukázalo jako křehké a má se přehodnotit.

## Co zbývá
1. **Tisk (pilíř A) doladit / přepsat přístup** — věrný vzhled + obrázky. Před dalším pokusem viz [chybový deník](../../chybovy-denik/tisk.md), ať se neopakují CH-001…008. Doporučení (CH-META): serverové PDF nebo dedikovaná tisková šablona, ne klonování živého DOMu.
2. **Import světa** (odložený follow-up).
3. **Reálné ověření** stažení ZIP v prohlížeči.
4. Po dotažení: zaškrtnout 14.7 v `roadmap2.md`, aktualizovat `funkce`/`napoveda`.
